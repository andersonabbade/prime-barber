# 💈 Prime Barber

Site profissional desenvolvido para uma barbearia moderna, com foco em design elegante, experiência do usuário e conversão de clientes via WhatsApp.

🔗 **Acesse o site:**  
https://prime-barber01.netlify.app/

---

## ✨ Sobre o projeto

O **Prime Barber** foi desenvolvido como um projeto prático para simular um site real de cliente.

O objetivo foi criar uma landing page completa, responsiva e pronta para uso comercial, com integração direta para agendamentos.

---

## 🎯 Funcionalidades

- Layout moderno e profissional
- Design totalmente responsivo (mobile, tablet e desktop)
- Menu interativo (hamburger no mobile)
- Seção "Quem somos" com apresentação da marca
- Catálogo de serviços com preços
- Botões de agendamento via WhatsApp
- Mapa integrado com localização
- Botão flutuante de contato (WhatsApp)
- Animações suaves com JavaScript

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Netlify (deploy)
